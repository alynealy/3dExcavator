# Excavator

## Overview
This arm has three main movements, like a human arm, to reach different heights and depths. By simply adjusting the length of the pistons, it positions the bucket very precisely, and the smart placement of the pivots gives it the power to lift heavy loads, just like a lever.



## Key Features
### 3-DoF Arm
The system utilizes three main pivots to reach various heights and depths, accurately mimicking the range of motion of a human arm.
### Variable Geometry
By adjusting the length of the pistons, the assembly modifies its internal angles to position the bucket with high precision.
### Mechanical Leverage
The strategic placement of pistons and pivots creates a "crowbar" effect, multiplying the input force to enable heavy lifting.

## Main Components
| Component | Description |
|-----------|-------------|
| **Boom** | The primary arm segment attached to the base; it handles the main vertical lifting. |
| **Stick (or Dipper)** | The secondary arm segment that provides the reach and "pulling" motion toward the machine. |
| **Bucket** | The end-effector designed for digging, scooping, or grabbing materials. |
| **Pistons (Actuators)** | The linear components that extend and contract to drive the movement of each segment. |
| **Pivot Pins (Hinges)** | The "bolts" or axes that allow rotation between the different structural parts. |
| **Arm Base** | The main support platform that holds the weight of the arm and serves as the primary anchor point. |
| **Tires** | The tires of the arm based so that the excavator could move. |
